# CJS B42 Body Location Compat

Keeps active custom B42 body locations and legacy loot selector IDs registered, and reports unsafe zombie visuals without deleting them.

## Mod Info

- Mod ID: `cjsB42BodyLocationCompat`
- Author: CJ Storrs
- Minimum Project Zomboid version: 42.0.0

## Layout

- `42/` contains the Build 42 mod files loaded by Project Zomboid.
- `Common/` or `common/` contains shared assets when this mod uses them.

## Notes

This repository tracks the Project Zomboid mod source used by the local development workspace.
